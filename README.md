# Halloween 31

A typographic "31" study for Halloween with a Castlevania blood moon aesthetic.

**[View Live](https://jwzy.github.io/halloween-31/)**

## Features

- Blood moon with pulsing red intensity
- Castle silhouette
- Animated bat flocks with motion blur
- Blood rain effect
- Ambient audio (wind, rain rumble) - auto-starts on interaction
- Ember glow typography using Cinzel Decorative

## Interactions

- **Click the castle** to release a swarm of bats that fly towards the camera
